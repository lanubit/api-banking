#API BANKING

## Description

Api created for work purposes, contains a service that shows nearby ATMs and exchange rate of the day

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deploy

```bash
# main deploy, project is deployed when commit to main branch
# For the deployment it is used: Github, Git Actions and Heroku
$ git push origin main
```
